자주 묻는 질문

궁금한 점이 있으신가요?

프로그램 자체는 50,000원 단 1회 결제로 평생 사용 가능합니다. 다만 AI 글쓰기를 위한 GPT(OpenAI) 또는 Gemini(Google) API 키는 직접 발급·사용하셔야 하며, 이 비용은 별도입니다. 글 한 편 기준 약 1~5원 수준으로 하루 3편 발행 시 월 1,000~5,000원 내외입니다.
사람이 직접 타이핑하는 것처럼 자연스럽게 동작하도록 설계되어 있습니다. 단, 하루 발행 횟수를 과도하게 설정하거나 동일한 글을 반복하는 경우 네이버 정책에 따라 제재 위험이 있으므로, 하루 1~3회 발행을 권장합니다.
프로그램이 로컬에서 실행되는 특성상, 컴퓨터가 켜진 상태여야 자동 발행이 가능합니다. 24시간 자동 운영을 원하신다면 컴퓨터를 상시 켜두시거나, 별도의 미니PC/맥미니 등을 활용하시는 것을 권장합니다.
라이선스 키 발급과 동시에 환불이 불가능합니다. 구매 전 기능과 사용 조건을 충분히 검토해 주세요. 구매 문의 시 궁금한 점을 먼저 질문하시면 성실히 답변 드립니다.
